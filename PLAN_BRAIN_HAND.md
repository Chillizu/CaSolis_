# CaSolis_ — 大脑+手脚架构计划书

> 📌 这是一个面向社区评审的架构计划。如果你被邀请看这个文件，请带着「这能行吗？」的怀疑眼光来读。

---

## 项目终极目标

**从零构建一个不依赖人类引导、自主在 Docker 沙箱中探索、持续学习、且每次行为都不同的 AI agent。**

不是另一个 LLM 应用。是探索「思考的本质 = 概率 + 最优选择」这个哲学假设在工程上能否落地。

## 三年走了什么弯路

### 起点（放弃）
用 Qwen/DeepSeek 做 SFT + GRPO 训练 → 很快发现这只是在「教一个 LLM 回答特定格式」，不是「从零构建思维」。

### 中间（15 种以上架构）
尝试了：GRU 7.6M、Mamba 7.3M→12.6M→19.7M、MTP（多 token 预测）、SlotMind（K-slot 工作记忆）、ModelCluster（MoE 专家集群）。

### 核心发现
所有架构的瓶颈**不是架构本身**，而是训练数据量和模型容量。
- ✓ 路由器（选择哪个专家/意图）可以在 46M 参数上学会
- ✓ 世界模型头（预测输出嵌入）可以在小模型上训练
- ❌ 小模型无论如何都无法可靠生成有效 bash 命令（不管是 CMD、INTENT 还是任何文本格式）
- ❌ 1500 条数据远不够
- ❌ CPU 上训练 100M+ 参数不现实（30 min/epoch 已是最快极限）

### 现在的转折
既然小模型**不能生成但能分类**，大模型**能生成但太**（GPU 要求或推理慢），那为什么不分开？

**大脑（小模型）做决策 → 手脚（大模型）做执行**

---

## 核心理念：为什么我们相信这能工作

将「思考/决策」与「执行/生成」分离为两个独立的系统。

```
大脑 (ModelCluster v3)         手脚 (QL-1.5B / 其他小 LLM)
──────────────────             ───────────────────
46M 参数, CPU 可训             1.5B+ 参数, Ollama 推理
学会: 选意图 (20选1)           负责: 把意图转成精确 bash
世界模型预测结果                执行命令并返回输出
评估/好奇心驱动                 不需要我们训练
```

## 为什么是大脑+手脚不是一个大模型

| 方案 | 优势 | 劣势 |
|------|------|------|
| 一个大模型 | 简单 | 需要 GPU 训练，违背"从零"理念 |
| 纯从小训练 | 完全自主 | 15M 以下无法生成有效命令 |
| **手+脑分离** | 各司其职，大脑小可训，手脚好用 | 两个系统需要协调 |

## 训练流程

### 阶段 1: 意图数据生成 (1-2 小时)
```
1. 从 clean-v3 (1452 条) + Docker 验证
2. 每条标注意图 (已有 intent_translator.py)
3. 扩展: 手动添加更多意图变体
4. 目标: 2000+ 条意图标注数据
```

### 阶段 2: 大脑微调 (2-3 小时)
```
1. 加载 ModelCluster v3 checkpoint
2. 替换 lm_head → intent_head (20 维输出)
3. 训练目标 = 意图分类交叉熵
4. 保持世界模型训练 (预测输出嵌入)
5. 5 epoch, CPU ~30 min/epoch
```

### 阶段 3: 手脚集成 (1 小时)
```
1. 启动 Ollama + Qwen2.5-1.5B
2. 意图 → prompt → 执行 → 解析
3. 集成到 Runner 循环
```

### 阶段 4: 闭环测试 (持续)
```
1. ModelCluster 选意图
2. Qwen 转 bash + 执行
3. 世界模型评估结果
4. 预测误差驱动下一轮探索
```

## 意图列表 (20 种)

## 好奇心驱动的探索循环

```
世界模型预测误差高
     ↓
路由器从「常规专家」切换到「创造专家 (E1)」
     ↓
创造专家输出: EXPLORE intent
     ↓
手脚: 从已知命令池随机选一个 + --help 运行
     ↓
新输出回来 → 世界模型再次预测
     ↓
误差还高 → 发现新领域 → 记下来
误差降低 → 学会了 → 正常利用
```

## 物理可行性

- **大脑**: 46M 参数, ~200MB RAM, CPU 推理 <5ms
- **手脚**: Qwen 1.5B, 需要 Ollama, ~3GB RAM, CPU 推理 ~200ms
- **总计**: 约 4GB + Docker, 30GB RAM 完全够
- **训练**: 大脑 46M 在 CPU 上 30min/epoch, 5 epoch = 2.5h
