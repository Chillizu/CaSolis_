# CaSolis_ — 自主 AI Agent 项目评审请求

## 项目梗概

**核心目标**: 从零构建一个能在 Docker 沙箱中自主运行的 AI agent，不依赖预训练模型，不依赖人类提示，实现「自己的思维 + 涌现创造 + 每次不一样的行为」。

**当前路线**: 
1. 自写 tokenizer → 自训练小模型（Mamba 架构）→ 命令生成 → 世界模型预测 → 多 Token 预测
2. 最新架构: SlotMind — K-slot 工作记忆（4 个独立思维槽 + 跨槽通信 + 路由门控）
3. 哲学基础: 「思考 = 概率 + 最优选择」，人类认知与机器预测本质相同

**物理限制**:
- CPU-only (Intel Core Ultra 9 185H, 16 核, 105°C 热限)
- 30GB RAM, ~24GB 可用
- Docker 可用
- 单人在家业余时间

### 四人不能干的事

当前模型无法做到：
1. ❌ 因果推理 — 不会「因为 A 所以 B」
2. ❌ 多步规划 — 不会「先查 help 再执行」
3. ❌ 好奇心驱动探索 — 世界模型误差没用来驱动行为
4. ❌ 泛化 — 没见过的命令就是没见过

### 我们要试的解决方向

SlotMind 架构试图通过以下方式解决：

**K 个独立思维槽**: 每个槽跟踪一条不同的「思路线」，并行演化
**跨槽通信**: 槽之间可以互相「看」，形成思路对话
**路由门控**: 学习决定哪个槽的输出用于行动（不是硬选，可以混合）
**创造机制** (需要加):
  1. 重组: 槽状态插值/杂交 → 新思路
  2. 意外驱动: 世界模型预测误差 → 探索信号
  3. 质量判断: 信心头 → 评估哪个创造性输出好

2. **架构方向**: K-slot 工作记忆合理吗？类似研究存在吗？
3. **被忽略的方向**: MoE? JEPA? 蒸馏? 我们错过了什么重要的？
4. **批评**: 最大的问题在哪？如果只做一件事，应该是什么？
5. **建议**: 基于我们的物理限制（CPU-only，30GB），下一步怎么走？
